🔒 Privacy Policy

MyOwnBank Terminal

Last updated: April 24, 2026

📋 Overview

MyOwnBank Terminal ("the App") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard information when you use our application.

📊 Information We Collect

Data Stored Locally

  • Terminal ID (generated automatically)
  • Transaction history (stored in Firebase)

Data Transmitted

  • Payment transaction data (amount, status, method)
  • Terminal configuration settings
  • Transaction timestamps

🎯 How We Use Your Information

The App uses Firebase Realtime Database to:

  • Synchronize payment data with management system
  • Store transaction history
  • Enable remote terminal management

💾 Data Storage

All data is stored in Firebase Realtime Database and is subject to Google's privacy policies: https://firebase.google.com/support/privacy

🔗 Third-Party Services

The App uses the following third-party services:

  • Firebase (Google LLC) - for real-time data synchronization
  • ML Kit (Google LLC) - for face detection (optional feature)

🔐 Data Security

We implement appropriate security measures to protect your data:

  • Encrypted data transmission
  • Secure Firebase authentication
  • No personal identification data collected

📱 Permissions

The App requires the following permissions:

  • Camera - for Face ID and card payment features
  • NFC - for contactless card payments
  • Internet - for Firebase synchronization

👶 Children's Privacy

The App is not intended for use by children under 13 years of age.

🔄 Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated revision date.
